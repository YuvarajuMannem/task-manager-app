# ✅ Task Manager App

A clean, simple, and fully functional Task Manager web application built with vanilla HTML, CSS, and JavaScript.

## 🚀 Features

- ➕ **Add** new tasks
- ✏️ **Edit** existing tasks
- 🗑️ **Delete** tasks
- ✅ **Mark tasks as complete/incomplete**
- 🔍 **Filter** tasks (All / Pending / Completed)
- 💾 **Persistent storage** (tasks saved in browser's localStorage)
- 📱 **Responsive design** (works on mobile & desktop)

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & animations
- **JavaScript (ES6+)** – Logic & interactivity
- **localStorage** – Data persistence

## 🧠 What Makes This Good

- Clean, modular **OOP** code (TaskManager class)
- No external dependencies or frameworks
- **XSS protection** (HTML escaping)
- Smooth UI with hover effects
- Keyboard support (Enter to add task)
- Confirmation before deletion
- Persistent data even after page refresh
